# Linux Fundamentals for Cybersecurity

## Overview

This repository contains my complete 14-day Linux Fundamentals learning journey focused on Linux administration, system troubleshooting, networking, and cybersecurity foundations.

The goal of this roadmap was to develop practical Linux skills that are essential for SOC Analysts, Cybersecurity Analysts, Ethical Hackers, and System Administrators.
